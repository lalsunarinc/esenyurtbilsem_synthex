## 🎮 Oyun Kontrolleri

| Tuş | Aksiyon |
|-----|---------|
| `W A S D` | Hareket (mekanum — her yön) |
| `Q / E` | Çapraz hareket |
| `SPACE` | Top al / Bente veya sarnıca bırak |
| `R` | Kaynaktan yeni top çağır (3sn bekleme) |
| `F` | Su kemerine asıl (sadece Endgame) |
| `← →` | Robot döndür |
| `TAB` | Strateji görünümü |

## 🚀 GitHub Pages ile Yayınlama

1. Bu repoyu aç → **Settings**
2. Sol menüden **Pages**'e tıkla
3. Source: **Deploy from a branch** → `main` → `/ (root)`
4. **Save** — birkaç dakika sonra `https://kullaniciadi.github.io/repo-adi` adresinde yayında!
